# Java 每日行程規劃程式 (Java-APP-project)

[![Language](https://img.shields.io/badge/%E4%B8%AD%E6%96%87-English-white?labelColor=2ea44f&style=flat)](README.md)

這是一個使用 Java 與 Swing GUI 撰寫的每日行程規劃與事務模擬應用程式，主要做為 Java 程式設計與圖形介面開發的練習專案。

## 專案概述

本應用程式透過讓使用者將每日空閒時間分配至五大生活面向，來模擬與管理日常行程：
- 學業 (Academic)
- 興趣 (Hobby)
- 運動 (Sport)
- 社交 (Social)
- 感情 (Relationship)

使用者可以定義任務、為不同類別指定權重，程式將進行每日行程模擬或追蹤相關統計數據。

## 功能特色

- **五大面向設定**：自訂五大生活核心的權重與細項。
- **行程檢視**：查看各類別的進度與分配時間。
- **排程模擬**：根據使用者設定的權重，從五大類別中每日隨機選擇 3 項任務。
- **執行與追蹤**：儲存模擬結果、計算達成率並記錄執行歷史。
- **動態視窗大小**：GUI 尺寸會根據使用者的螢幕大小自動縮放適應。

## 系統架構理念

1. **模擬類別**：
   - 參數：每日空閒小時數、事項、事項比重、五大項目比重
   - 輸出：主程式介面、圖表、資料庫
   - 函式：每日五選三隨機選擇、所選結果儲存、遞迴執行選擇、結果統計
2. **項目類別 (五大面向)**：
   - 參數：比重、子項目陣列
   - 輸出：子項目新增介面、子項目條目
3. **子項目類別 (例如：學業->電腦網路)**：
   - 參數：比重、已做時數、成功條件時數、註解
4. **已執行項目類別**：計算各項子類別執行時間與儲存紀錄。
5. **未執行項目類別**：執行率計算與資料讀取。
6. **主程式類別**：首次登入設定、每日任務挑選、修改參數與結果統計。

## 如何執行

1. 請確保您的環境已安裝 Java 開發套件 (JDK)。
2. 編譯並執行 `GUI.MainGUI` 類別：
   ```bash
   javac GUI/*.java
   java GUI.MainGUI
   ```

## 模擬所需輸入參數

- 實驗總天數
- 每天工作時數 (預設 8 小時)
- 各項權重
- 每個細項成功所需時數 (加總不得超過實驗總時間)

*註：各細項分配到的時數公式為：`該細項權重 / 該細項所屬面向的全部細項總權重 * 該面向權重 / 面向總權重 * 每日時間`*
